Uptime monitoring packed into a single Go binary with embedded SQLite.

Asura monitors your infrastructure from a single Go binary backed by SQLite.
No Postgres. No Redis. No Node.js. Just scp a binary and go.

Quick Start

git clone https://github.com/y0f/Asura.git && cd Asura && sudo bash install.sh

Handles everything. Go install, build, systemd service, admin key. Takes a couple minutes on a fresh Ubuntu box.

Web UI

Server-rendered Go templates (templ), progressively enhanced with HTMX and Alpine.js, styled with Tailwind CSS. Configure monitors, build assertion groups, manage notifications, browse incidents, and set up status pages. all from the browser. You can also skip the UI entirely and use the REST API.
